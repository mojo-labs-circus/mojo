## Declaration

**Name:** mojo-framework-dev-team

**Parent collective:** mojo-labs

**Goal:** Design and build the Mojo framework. Mojo Circus is the infrastructure
layer that runs framework instances on real hardware — this team designs both.
By operating as a collective while building the framework, this team dogfoods it:
every design decision gets tested against real use, every gap surfaces as a real problem.

## Operating Principles (Stubs)

Working assumptions until the framework sessions formalise them.

**Sovereignty** — individual data is never shared without explicit consent. *(Session 01)*

**Open source first** — reach for open source before writing from scratch. MIT licence:
include copyright notice, acknowledge source, done.

**Technically capable audience first** — default to powerful and flexible over easy.
Accessibility follows once the fundamentals are solid.

**Accountability** — every action traces to a member. AI actions trace to their human
owner. The chain is never broken. *(Session 06)*

**Composability** — collectives nest. The model is the same at every scale. *(Session 01)*
