## Cortex OS (agentic.james, Instagram)

Multi-agent Claude Code orchestration tool. Agent-to-agent comms, scheduled tasks, accessible from phone. Described by its creator as "multi-agent Claude Code from your phone."

Locked behind a $97/month Skool.com course — content business, not an open platform.

**How it differs from MojOS:** Cloud-dependent, no private compute, no OS layer, no fleet coordination, no self-hosted hardware. It is a workflow tool built on top of existing cloud AI. MojOS is infrastructure that owns the layer underneath.

**Verdict:** Mid-tier overlap on the orchestration concept. Not a serious competitive threat to the core MojOS vision.
